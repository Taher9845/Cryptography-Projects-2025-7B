About

  • Key Generation: Each user has an ECC P-256 key pair.
  • Authenticated Exchange: A mini-CA ECDSA-signs user public keys.
  • Key Derivation: ECDH secret -> SHA-256 based KDF to 256-bit AES key.
  • Encryption: AES-CBC with PKCS#7 padding.
  • Obfuscation: Deterministic block permutation using SHA-256(key||iv||i).
